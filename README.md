# Laboratorio 3 - Arquitecturas de Software & Integración Continua (Bootstrap 5)

Este proyecto es una práctica de diseño y maquetación web empleando HTML5 y **Bootstrap 5**. El objetivo de este laboratorio fue estructurar y decorar de manera interactiva una página web que explique diferentes conceptos teóricos.

## Características Agregadas:
- **Comparativa de Arquitecturas**: Diseño en tarjetas (Cards) explicando las ventajas y desventajas de Monolitos, Arquitectura Hexagonal y Microservicios.
- **Integración Continua (CI)**: Sección informativa sobre las prácticas y ventajas de herramientas CI/CD en el desarrollo ágil.
- **Presentación de Diapositivas**: Se integró un *Carousel* manual responsivo (las diapositivas no avanzan solas, dando control durante presentaciones), cargando láminas PNG para presentar un tema secuencialmente con sus indicadores.
- **Navegación Intuitiva**: Menú superior (`Navbar`) implementado de forma *sticky* al inicio, el cual permite un desplazamiento inmediato mediante enlaces a anclas (`#arquitecturas`, `#comparativa`, `#integracion` y `#presentacion`).

## Tecnologías Utilizadas:
- HTML5
- CSS (Bootstrap 5 CDN)
- Bootstrap Icons
